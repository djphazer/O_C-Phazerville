# BitBeat

(TODO: screenshot)

**BitBeat** is an applet port of the Byte Beats algorithm from the [**Viznutcracker, Sweet!**](Viznutcracker-sweet) stock firmware App. There are two independent signal generators.

The CV inputs can be freely assigned for modulation by pressing AuxButton with the target parameter selected for editing. A small "1" or "2" will appear. Each input can be assigned to multiple destinations.

### I/O

|        |          1/3         |         2/4           |
| ------ | :------------------: | :-------------------: |
| TRIG   | Channel 1 Reset      | Channel 2 Reset       |
| CV INs | Assignable           | Assignable            |
| OUTs   | Ch 1 signal          | Ch 2 signal           |

### UI Parameters
For each channel:
* Equation selection (0 to 15)
* Speed
* Pitch
* Step mode
* Parameter 1
* Parameter 2
* Parameter 3
* Loop mode
* Loop start
* Loop end
